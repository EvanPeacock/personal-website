    Projects

    MultiRecorder

    MultiRecorder is a simple utility that was originally created to streamline the testing process at my current job at Piasecki Aircraft. It allows users to remotely control multiple OBS instances and BlackMagic HyperDeck recorders from a simple GUI. It is written in Python, and uses DearPyGui for the user interface. It uses OBS WebSockets and the BlackMagic HyperDeck REST API to control connected devices.


    SoundLogic

    SoundLogic is a web application that I and a group of classmates created for our senior project at Rowan University. It is a music recommendation and sharing platform, with the humorous twist that users cannot like music, only dislike. It was written in Python using the Django framework. It also uses the Spotify API to fetch music data, and both Spotify and Google can be used for third-party authentication. Other features include user profiles, following other users, creating playlists, and more. I was responsible for many aspects of the project, but my largest contributions were the user interface, CSS styling, and Spotify API integration.


    Incident Reports

    Incident Reports is a cross-platform mobile application that I and a group of classmates created for our Software Engineering course at Rowan University. It is a simple application that allows users to report any safety incidents they see on campus or elsewhere. Incidents could be assigned a category, severity level, and location via GPS. The app was written in Flutter and Dart and uses Firebase for backend services. Other features include user authentication, a shortcut to call a set emergency contact, and a map and list view of incidents. I was responsible for the user interface, including the design and implementation of the app's pages and navigation. I also contributed to the app's backend, including database integration with the GUI and user authentication.
